# AceCamp field map

## source markdown sections

1. 基础信息
2. 要点（页面原文）
3. 正文
4. 智能追问
5. 标签
6. 专家与作者信息

## manifest/index mapping

- article_date -> 文章发布日期（date）
- provider -> acecamptech
- article_id -> 详情页ID
- article_title -> 标题
- crawl_date -> 抓取日期
- source_url -> 详情页URL
- record_time -> 记录时间（Asia/Shanghai）
- source_path -> source文件相对路径
- rendered_path -> rendered文件相对路径
- status -> ok
- content_type -> 类型（如纪要）
- industry -> 行业字段
- tags -> 标签数组
- author -> 发布人
